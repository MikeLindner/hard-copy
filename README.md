## Hard Copy
A command to quickly output text files to plain paper printer.

### Usage:
``` hc <ascii-filename>```

### Installation:
```sudo cp hc.sh /usr/local/bin/hc && sudo chmod +x /usr/local/bin/hc```

### Demonstration:
```hc hc.sh```

### Files:
```/etc/enscript/enscript.cfg```

### Reference:
https://tashian.com/guides/print-out-your-code/
